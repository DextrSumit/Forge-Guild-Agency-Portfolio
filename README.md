# Forge Guild Agency Portfolio

A premium, interactive digital portfolio for Forge Guild, showcasing services across Web Development, Video Production, Brand Identity, and Graphic Design.

## 🚀 Overview

Forge Guild Agency Portfolio is built to provide an immersive and highly dynamic user experience. It leverages modern web technologies to deliver sleek micro-animations, smooth transitions, and high-quality aesthetic visuals. 

### Key Sections:
- **Home:** High-impact landing page with workflow insights and statistics.
- **Services:** Detailed overview of agency offerings including Video Editing, Graphic Design, Web Development, and Brand Identity.
- **Projects:** Filterable archive of delivered projects across different disciplines.
- **Our Team:** Showcase of the core team members driving the agency's success.
- **Contact Us:** A sleek interactive form for onboarding and proposals.

## 🛠 Tech Stack

- **Framework:** [React 19](https://react.dev/) + [Vite](https://vitejs.dev/)
- **Routing:** [React Router DOM](https://reactrouter.com/)
- **Styling:** Vanilla CSS + [TailwindCSS 4](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/) & [GSAP](https://gsap.com/)

## ⚙️ Getting Started

Follow these steps to set up the project locally.

### Prerequisites

- Node.js (v18 or higher recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/DextrSumit/Forge-Guild-Agency-Portfolio.git
   cd Forge-Guild-Agency-Portfolio
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to the local URL provided in your terminal (usually `http://localhost:5173`).
